# Chat App

Chatroom app made with NodeJS and Socket.io.  
Runs multiple chat feeds simultaneously in the browser.
Tested with Mocha & ExpectJS.

## Production Site
[Demo on Heroku](url:https://sheltered-meadow-90758.herokuapp.com/)

### Or Run Locally
* `git clone git@github.com:AliSkinner/chat-app.git`
* `cd chat-app`
* `npm install`
* `npm start`
* `http://localhost:3000/`
